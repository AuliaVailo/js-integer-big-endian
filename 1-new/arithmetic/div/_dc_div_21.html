Algorithm 3.3 Divide-and-conquer division (2 by 1)

Input

              |  :  |  :  |               |  :  |
               -----------                 -----

Output

The quotient floor( A/B ) and the remainder A mod B.

Complexity

T(n) = 2T'(n/2) + K
